# AI剧本创作字段映射 (AI Script Writing Fields)

## 1. 基础设定 (Basic Settings)
- **核心参数 (Core Params)**
    - 题材设定 (Genre)
    - 风格倾向 (Style Tendency)
    - 核心受众 (Target Audience)
    - 剧本叙事 (Narrative Style)
    - 标签 (Tags)
- **规格参数 (Specs)**
    - 生成集数 (Episode Count)
    - 单集时长 (Episode Duration)
    - 深度记忆引擎 (Deep Memory Engine)
- **内容设定 (Content Settings)**
    - 作品名称 (Title)
    - 世界观设定 (World View)
    - 核心金手指 (Golden Finger)
    - 主线剧情 (Main Plot)
    - 角色档案 (Character Info)
    - 作品简介 (Synopsis)
    - 创作要求 (Requirements)

## 2. 大纲生成 (Outline Generation)
- **大纲列表 (Outline List)**
    - 集数 (Episode Number)
    - 标题 (Title)
    - 概要 (Summary)
    - 状态 (Status)
- **元数据设定 (Meta Data)**
    - 场景 (Scene)
    - 角色 (Roles)
    - 道具 (Props)
    - 冲突 (Conflict)
    - 黄金3秒 (Hook)
    - 关系图谱 (Relation)

## 3. 剧本撰写 (Script Writing)
- **编辑器 (Editor)**
    - 剧集标题 (Chapter Title)
    - 正文内容 (Content)
    - 字数统计 (Word Count)
- **脚本格式标签 (Script Tags)**
    - [EP] 剧集
    - [INFO] 信息
    - [ROLE] 角色
    - [PROP] 道具
    - [SCENE] 场景
    - [STORY] 大纲
    - [SHOT] 分镜
    - [CAM] 镜头
    - [SFX] 音效
    - [TIME] 时长
    - [BGM] 背景音乐
- **辅助功能 (Assistants)**
    - AI 助手 (AI Chat)
    - 五感描写填充 (Five Senses)
    - 润色/续写/扩写 (AI Actions)
