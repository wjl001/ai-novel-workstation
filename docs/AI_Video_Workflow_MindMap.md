# AI视频创作工作流

## 1. 作品初始化 (Project Init)
- **基本信息**
    - 剧集总数: 10集
    - 作品类型
    - 作品简介
        - 角色档案
            - 名字
            - 设定
- **故事大纲**
    - 标题
    - 概要
    - *逻辑: 基于概要扩充* -> 指向剧本创作

## 2. 剧本创作 (Script Writing)
- **上：剧集脚本 (Episode Script)**
    - 标题
    - 剧情内容
    - 主体要素 (Entities)
        - 角色 (Character): 名称, 描述, 音色
        - 道具 (Props): 名称, 描述
        - 场景 (Scene): 名称, 描述
- **下：分镜脚本 (Storyboard Script) (交付)**
    - 1. 分镜编号
    - 2. 标题
    - 3. 总时长 (10-15s)
    - 4. 主体画面 (Key Visual)
    - 5. 镜头列表 (Shots) (多个)
        - ① 编号
        - ② 画面描述
        - ③ 音效
        - ④ BGM

## 3. 资产生成 (Asset Generation)
- **1> 主体生成**
    - 角色图片
    - 道具图片
    - 场景图片
- **2> 分镜细化**
    - *输入: 基于分镜脚本*
    - 首帧图描述
    - 画面详细描述

## 4. 视频合成 (Video Synthesis)
- **3> 视频生成**
    - 封面生成
        - 来源: 剧情 + 主体
    - 分镜视频 (多个)
        - 来源: 首帧图 + 画面描述 + 主体
        - 步骤
            - 生成首帧图
            - 生成视频片段
    - 背景音乐
        - 来源: 剧情情感分析
- **4> 最终合成**
    - 输出: 纯视频文件
